Introduction of OSI

Go To Page

The OSI (Open Systems Interconnection) model is a way to explain how data travels from one device to another over a network. It is divided into seven layers, and each layer has a specific job in sending and receiving information. These layers help organize how software, hardware, and networking systems work together to make communication possible. By using the OSI model, network engineers can better understand, design, and fix problems in networks because they know exactly which layer is responsible for each part of the communication process.

OSI lAYERS:

  1. Physical
  2. Data Link
  3. Network
  4. Transport
  5. Session
  6. Presentation
  7. Application

OSI Layers and Their Functions

Layers Purpose
Physical Convert data into bits, sends over cables
Data Link Working with switches adding MAC address to the data
Network Working with router adding IP address to the data
Transport Provide reliability to data by adding port numbers
Session Set-up and tear down the communication between devices
Presentation Formats, encrypts, and compresses data
Application Allows users to interact with the network; working with protocols
